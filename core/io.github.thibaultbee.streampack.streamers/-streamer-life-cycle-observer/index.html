Add DefaultLifecycleObserver to a streamer.

You will not have to call IStreamer.release when application is destroyed nor to to call IStreamer.stopStream when application goes to background.

To use it, call:

  • lifeCycle.addObserver(StreamerLifeCycleObserver(streamer))
