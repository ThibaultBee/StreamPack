startStream

@RequiresPermission(["android.permission.RECORD_AUDIO", "android.permission.CAMERA"]) suspend fun startStream(ip: String, port: Int): Unit

Overrides ILiveStreamer.startStream

Connect to an SRT server and start stream. Same as calling connect, then startStream. To avoid creating an unresponsive UI, do not call on main thread.

Parameters

ip - server ip

port - server port

Exceptions

Exception - if connection has failed or configuration has failed or startStream has failed too.
