fun configure(audioConfig: AudioConfig, videoConfig: VideoConfig)
Content copied to clipboard

Configures both video and audio settings. It is the first method to call after a BaseCameraStreamer instantiation. It must be call when both stream and capture are not running.

Use CameraStreamerConfigurationHelper to get value limits.

If video encoder does not support VideoConfig.level or VideoConfig.profile, it fallbacks to video encoder default level and default profile.

Inside, it creates most of record and encoders object.
