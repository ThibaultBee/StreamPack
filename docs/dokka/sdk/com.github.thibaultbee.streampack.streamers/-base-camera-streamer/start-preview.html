startPreview

@RequiresPermission(["android.permission.RECORD_AUDIO", "android.permission.CAMERA"]) fun startPreview(previewSurface: Surface, cameraId: String = "0"): Unit

Starts audio and video capture. BaseCameraStreamer.configure must have been called at least once.

Inside, it launches both camera and microphone capture.

Parameters

previewSurface - Where to display camera capture

cameraId - camera id (get camera id list from Context.getCameraList)

Exceptions

StreamPackError - if audio or video capture couldn't be launch
